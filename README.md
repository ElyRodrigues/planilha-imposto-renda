# Planilha de Imposto de Renda

Este repositório contém uma planilha em Excel para organizar e informar rendimentos financeiros, útil para a declaração de imposto de renda.

## Conteúdo

- Planilha em formato `.xlsx`
- Campos para preenchimento de rendimentos, deduções e totais
- Filtros para facilitar a visualização por ano ou tipo de rendimento

## Como usar

1. Baixe a planilha
2. Preencha os campos com seus dados financeiros
3. Utilize os filtros para gerar relatórios personalizados
